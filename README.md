# TravelGuide - Site de Recomendação de Viagens

Um site moderno e interativo para recomendação de destinos de viagem, desenvolvido com HTML, CSS e JavaScript puro.

## 🚀 Funcionalidades

### ✅ Requisitos Atendidos (20 pontos)

- **URL pública do repositório GitHub** (1 ponto)
- **Páginas funcionais**: Início, Sobre Nós e Contato (3 pontos)
- **Barra de navegação** responsiva e interativa (6 pontos)
- **Introdução na página inicial** com hero section atrativo (1 ponto)
- **Elementos da página Sobre nós** com estatísticas e valores (2 pontos)
- **Formulário de e-mail** na página de contato com validação (1 ponto)
- **Recomendação de praia** com duas imagens e descrições (2 pontos)
- **Recomendação de templo** com duas imagens e descrições (2 pontos)
- **Recomendação por país** com duas imagens e descrições (2 pontos)

## 🎨 Características do Design

- **Design Responsivo**: Funciona perfeitamente em desktop, tablet e mobile
- **Interface Moderna**: Gradientes, sombras e animações suaves
- **Navegação Intuitiva**: Menu hambúrguer para dispositivos móveis
- **Experiência Interativa**: Efeitos de hover, animações e transições
- **Acessibilidade**: Cores contrastantes e navegação por teclado

## 📁 Estrutura do Projeto

```
├── index.html          # Página inicial
├── about.html          # Página Sobre Nós
├── contact.html        # Página de Contato
├── styles.css          # Estilos CSS
├── script.js           # Funcionalidades JavaScript
└── README.md           # Documentação
```

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica e acessível
- **CSS3**: Estilos modernos com Flexbox e Grid
- **JavaScript ES6+**: Funcionalidades interativas
- **Font Awesome**: Ícones vetoriais
- **Google Fonts**: Tipografia Poppins

## 🚀 Como Usar

1. Clone ou baixe o repositório
2. Abra o arquivo `index.html` em um navegador web
3. Navegue pelas diferentes páginas usando o menu
4. Teste o formulário de contato
5. Explore as recomendações de destinos

## 📱 Funcionalidades JavaScript

- **Navegação Mobile**: Menu hambúrguer responsivo
- **Scroll Suave**: Navegação fluida entre seções
- **Animações**: Efeitos de entrada e hover
- **Formulário Interativo**: Validação e feedback em tempo real
- **Sistema de Recomendações**: Base para futuras expansões
- **Efeito Parallax**: Movimento sutil no hero section

## 🎯 Seções do Site

### Página Inicial
- Hero section com call-to-action
- Seção de recomendações com três categorias:
  - Destinos de Praia (Caribe e Brasil)
  - Destinos Espirituais (Japão e Tailândia)
  - Destinos por País (França e Itália)

### Página Sobre Nós
- Missão e valores da empresa
- Estatísticas animadas
- Grid de valores com ícones
- Informações sobre a equipe

### Página de Contato
- Formulário completo com validação
- Informações de contato
- Seção de perguntas frequentes
- Feedback visual para envio
